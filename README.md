# Kitchen_Calculator
Desktop app converting units for products used in cooking recipes.

## Motivation
My first JavaFX app.

## Functionality
Based on a chosed product the app will convert the amount of initial unit to a selected different unit.

## Technology
Java SE 8 + JavaFX
